# Portafolio Web

**Descripción:**
Este proyecto es una prueba de diseño y desarrollo web, implementado como portafolio personal. Incluye una interfaz responsive, estilos modernos con CSS y funcionalidades básicas con JavaScript.

---

## Características
- Diseño mayormente responsive (adaptable a móviles y escritorio).
- Secciones: Inicio, Sobre mí, Proyectos, Contacto.
- Animaciones y efectos visuales con CSS y JavaScript.
- Estructura semántica de HTML5.

---

## Tecnologías Utilizadas
- **HTML5**: Estructura del contenido.
- **CSS3**: Estilos, animaciones y diseño responsive.
- **JavaScript**: Interactividad y manipulación del DOM.
